# Co-Working Space Booking System

## Purpose

The Co-Working Space Booking System is designed to facilitate the reservation of desks in a co-working environment. It allows users to book individual desks or team desks for specified time periods, calculates the booking cost based on membership tiers, and maintains a revenue dashboard for tracking earnings categorized by membership types.

---

## How to Use

### Booking a Desk

1. **Select Desk Type**: Choose between "Individual Desk" or "Team Desk" from the dropdown menu.
2. **Select Membership Type**: Choose your membership tier from "Basic", "Premium", or "Executive".
3. **Booking Time**: Enter the date and time for which you want to book the desk.
4. **Submit**: Click on the "Book Desk" button to confirm your booking.

### Price Calculation

- **Individual Desks**:
  - **Basic Membership**: $10 per hour
  - **Premium Membership**: $15 per hour
  - **Executive Membership**: $20 per hour

- **Team Desks**: $25 per hour

- **Discount**: If you book for more than 3 hours, you will receive a 10% discount on the total booking cost.

### Revenue Dashboard

- The dashboard displays the total revenue collected over a certain period, categorized by membership types.

## Setup

To run the application locally:

1. Clone the repository from GitHub:

   ```
   git clone https://github.com/IgwehAkonuche/web3bridge_assesment.git
   ```

2. Navigate into the project directory:

   ```
   cd co-working-booking-system
   ```

3. Open `index.html` in your web browser.
